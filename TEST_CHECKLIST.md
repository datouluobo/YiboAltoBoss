# YiboAltoBoss 测试清单

用途：用于逐条验证 `YiboAltoBoss` 当前版本的功能、交互、数据准确性与稳定性。

建议使用方式：
- 每完成一项，就在对应条目前打勾，或在“结果记录”里写下现象。
- 如果发现问题，尽量补充“复现步骤 / 当前角色 / 当前服务器 / 是否重载过 / 是否跨角色”。

记录约定：
- 状态：`[ ] 未测` `[-] 进行中` `[x] 通过` `[!] 异常`
- 结果记录：可直接填写“通过 / 不通过 / 偶现 / 待复测”等。
